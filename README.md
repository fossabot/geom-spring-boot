# Geom

与市面上大部分的后台管理服务不同，本项目的目的是提供一组直接部署的微服务组件。使用者无需二次开发，也不需要了解内部实现细节。服务对外提供有限的接口并由文档对其进行说明，使用者通过SDK与接口进行交互。
